# Banking Management System

A console-based Banking Management System built in C++. This project demonstrates foundational Object-Oriented Programming (OOP) principles and persistent data storage using binary file handling.

## Features

* Create new bank accounts with initial deposits
* Perform secure deposits and withdrawals
* Check current account balances
* Persistent data storage (data is saved across sessions in `bank_data.dat`)
* Interactive command-line menu

## Prerequisites

To compile and run this project, you will need a C++ compiler installed on your system, such as GCC (MinGW for Windows) or Clang.
